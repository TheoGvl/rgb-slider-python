# rgb Color Mixer

Ένα εντυπωσιακό και διαδραστικό εργαλείο για να φτιάχνεις τα δικά σου χρώματα! Το έφτιαξα για να δοκιμάσω πώς δουλεύουν τα Sliders (μπάρες) και πώς μπορεί μια εφαρμογή να αντιδρά σε **πραγματικό χρόνο**, χωρίς να χρειάζεται να πατάς συνέχεια κάποιο κουμπί "Υπολογισμού".

## Τι ακριβώς κάνει;
Έχεις μπροστά σου 3 μπάρες: μία για το Κόκκινο (Red), μία για το Πράσινο (Green) και μία για το Μπλε (Blue). 
* Καθώς σέρνεις τις μπάρες (από το 0 έως το 255), το μεγάλο κουτί στην οθόνη αλλάζει χρώμα αστραπιαία.
* Ταυτόχρονα, το πρόγραμμα υπολογίζει και σου εμφανίζει τον ακριβή κωδικό **HEX** (π.χ. `#FF0000`) και το **RGB**, ώστε να μπορείς να πάρεις το χρώμα και να το χρησιμοποιήσεις στο δικό σου κώδικα ή σε κάποιο site!

## 🛠️ Τεχνολογίες & Τι έμαθα
Το project είναι γραμμένο σε **Python**, με την καταπληκτική βιβλιοθήκη **Flet**. 

**Τι κράτησα από αυτό το project:**
* Τη χρήση του `on_change` (αντί για `on_click`), που επιτρέπει στο πρόγραμμα να καταλαβαίνει την παραμικρή κίνηση στη μπάρα και να ανανεώνει την οθόνη.
* Το πώς να χρησιμοποιώ τα `Containers` της Flet για να φτιάχνω μοντέρνα σχήματα με στρογγυλεμένες γωνίες (`border_radius`).
* Λίγη μαγεία της Python: Μετέτρεψα τους απλούς αριθμούς (0-255) σε δεκαεξαδικό κώδικα HEX χρησιμοποιώντας f-strings (`f"#{r:02x}..."`).
